Program ini adalah website sederhana yang digunakan untuk mengelola produk sepatu. Di dalamnya terdapat beberapa fitur interaktif seperti dark mode, beli produk, dan wishlist.
1. Dark Mode
Mengatur tampilan website agar bisa berubah ke mode gelap dan disimpan di browser. Statusnya disimpan di localStorage Jadi walaupun halaman di-refresh, tema tetap tersimpan

2. Fitur Beli (Mengurangi Stok)
Program mengambil jumlah stok dari tombol "beli" jika di klik tombolnya maka angka stoknya akan semakin berkurang 

3. Fitur Wishlist
Nama produk dimasukkan ke dalam array,Disimpan ke localStorage jika produk sudah ada di wishlist, tidak bisa ditambahkan lagi Jumlah wishlist di navbar otomatis bertambah

4. Modal Wishlist
Data wishlist diambil dari localStorage, Ditampilkan dalam bentuk list di dalam modal Jika kosong muncul tulisan "Wishlist kosong" lalu jika di klik tomboh "hapus/kosongkan wishlist" maka modal wishlist akan otomatis kosong.
